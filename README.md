# KVage

 terminal cli for saving and retrieving key-value pairs in a file

 ## Usage

 List all key-value pairs

 ```bash
 kv list
 ```

Save a key-value pair

```bash
kv set <key> <value>
```

Update a key-value pair

```bash
kv up <key> <value>
```

Get a value by key

```bash
kv get <key>
```

Delete a key-value pair

```bash
kv rm <key>
```
